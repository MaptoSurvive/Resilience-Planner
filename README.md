# Resilience Planner App

My first project to help people prepare for emergencies.

## How to Run
1. Install [Node.js](https://nodejs.org)
2. Download this project
3. Open terminal in the project folder
4. Run:
   ```bash
   npm install
   npm run dev
   ```

## Features
- Add preparedness tasks
- Simple checklist
- Works on phones
